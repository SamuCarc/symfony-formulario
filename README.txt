
- Para crear la base de datos ir al archivo ".env" del directorio root en la linea 32 y poner los datos de la Base de datos
- Para crear la tablas de las entidades ir a la terminal y ejecutar "php bin/console doctrine:database:create" 
y php bin/console doctrine:database:update --force"
- Inicar servidor local con "symfony server:start" y abrir localhost:8000
